# Onassis - Free Bootstrap Template

A responsive single-page website template built with **HTML5**, **CSS3**, and **Bootstrap 5**.

## Preview

![Hero Section](images/hero-bg.png)

## Features

- Fully responsive design (Desktop, Tablet, Mobile)
- Built with Bootstrap 5 (via CDN)
- Clean and minimal UI
- No JavaScript required
- Beginner-friendly code structure

## Sections

- **Navbar** – Responsive navigation bar with hamburger toggle
- **Hero** – Full-width background image with overlay text
- **Welcome** – Introduction text with bold highlights
- **Our Services** – 4-column grid with SVG icons (London, Berlin, Paris, Tokyo)
- **Some Charts** – CSS-only circular progress rings
- **Quote Banner** – Orange motivational banner
- **About Us** – Dark section with description text
- **Contact Us** – Contact info with email, phone, and social links
- **Contact Form** – Styled form with name, email, subject, and message fields
- **Footer** – Simple copyright bar

## Technologies Used

| Technology | Purpose |
|------------|---------|
| HTML5 | Page structure |
| CSS3 | Custom styling |
| Bootstrap 5 | Responsive grid, navbar, forms, buttons |
| Bootstrap Icons | Social media icons |
| Google Fonts | Playfair Display, Open Sans |

## How to Run

1. Clone the repository:
   ```bash
   git clone https://github.com/your-username/onassis-template.git
   ```
2. Open `index.html` in any web browser.

> No build tools or server setup required.

## Responsive Breakpoints

| Breakpoint | Screen Width |
|------------|-------------|
| Desktop | > 992px |
| Tablet | 768px – 991px |
| Mobile | < 768px |
